# JUSTICESphere

JUSTICESphere is a unified digital platform developed to improve crime awareness, legal assistance, and secure evidence management. The system helps users report incidents, access legal support, manage evidence securely, and stay informed about crime-related activities.

## Features

- User registration and login
- Crime reporting system
- Secure evidence upload and management
- Crime awareness and alerts
- Legal assistance support
- Admin dashboard for monitoring and management
- Location-based crime information
- Secure file handling using encryption techniques
